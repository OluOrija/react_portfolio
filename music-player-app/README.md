# Music Player App

A simple music player app built with React and Material UI.

## Features

- Play/pause controls
- Next and previous track navigation
- Playlist management

## Technologies Used

- React
- Material UI

## Getting Started

### Prerequisites

- Node.js (and npm or yarn)

### Installation

1. Clone the repository:

   ```bash
   git clone <repository-url>
   cd music-player-app

2. **Install dependencies:**

    ```bash
    npm install
    ```

3. **Run the App**

    ```npm start
    ```
